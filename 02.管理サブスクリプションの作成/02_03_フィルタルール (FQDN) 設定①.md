# フィルタルール (FQDN) 設定①

Azure Firewall にフィルタルールを設定します。カスタマイズしやすいよう、フィルタルールの作成は 2 段階に分けて行っています。

- 本スクリプトでは、Windows Update や KMS など、多くの Windows/Linux システムで共通的に穴開けすることになる FQDN を開放しています。
